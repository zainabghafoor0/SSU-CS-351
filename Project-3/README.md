# Project 3 – WebGL Procedural Geometry  
CS 351 – Computer Graphics  
Zainab Ghafoor
This project explores procedural geometry in WebGL by creating shapes entirely in shaders.  
Each version of the program builds on the last, progressing from simple wireframe geometry  
to animated and colored procedural stars.
